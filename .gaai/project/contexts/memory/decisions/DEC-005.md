# DEC-005 — Service Account Auth for Google Sheets (Not OAuth)

## Context

n8n workflows that read/write Google Sheets in unattended automation contexts encounter OAuth token expiry, causing silent failures. A more stable auth method is required for production automations.

## Decision

Google Sheets integration in n8n uses Service Account credentials, not OAuth.

## Impact

- All n8n Google Sheets nodes must be configured with Service Account credentials
- OAuth must not be used for unattended automations (token expires, breaks silently)
- Service Account setup required once per Google Cloud project; credential JSON stored in n8n credentials
