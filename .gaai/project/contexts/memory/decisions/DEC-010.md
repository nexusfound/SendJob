# DEC-010 — Resend for Transactional Email (Not Gmail)

## Context

SendJob sends transactional emails for job confirmations and notifications. Gmail was initially noted as the email tool but Resend is the confirmed production email provider.

## Decision

Resend is the transactional email provider. Sender: `dispatch@sendjob.app`. Gmail is not used for transactional email in production.

## Impact

- All n8n email nodes use Resend (HTTP Request to Resend API or Resend n8n node)
- Per-client custom sender domain available as an upgrade
- Gmail references in older memory/conventions are incorrect — Resend is authoritative
